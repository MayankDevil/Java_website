two dimensions array

  • 2D Array is method to store array inside multiple array.
  • it array have two dimentaional array (row and column).
  • 2d array is dynamic in Java.
  • default value is set.

explain 

 [0]	[1]		[2]
  |		 |		 |
{(1)	(2)		(3)}    [0]

{(4)	(5)		(6)}    [1]

{(7)	(8)		(9)}    [2]

[0][1] = 2
[2][2] = 9

in 2Dimenstional Array we have row inside column

  1. syntax : declartion 

    2. datatype array_name[][];

  2. syntax : memory allocation 

    3. array_name = new dataype[Column_Size][Row_Size];

  3. syntax : assign 

    4. array_name[index][index] = value;

syntax : insalization = ( assign at declaration time ) 

datatype array_name[][] = { {value, ...}, ... };

example 

class Test
{
	public static void main(String args[])
	{
		int a[][];

		a = new int[3][3];

		a[0][0]	= 1;
		a[0][1]	= 2;
		a[0][2]	= 3;

		a[1][0]	= 4;
		a[1][1]	= 5;
		a[1][2]	= 6;

		a[2][0]	= 7;
		a[2][1]	= 8;
		a[2][2]	= 9;

		for (int o = 0; o < a.length; o++)
			for (int i = 0; i < a[o].length; i++)
				System.out.println("index["+o+"]["+i+"]"+a[o][i]);

	}
}
