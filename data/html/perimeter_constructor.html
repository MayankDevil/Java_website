perimeter constructor

  • perimeter constructor is same like constructor but they not call without pass arguments.
  • if we have only perimeter constructor so we not create object without give arguments.

syntax 

ClassName (datatype argument_name) { // code }

example 

class Hello
{
	Hello(boolean args)
	{
		System.out.println(" if pass arguments than it is execute");
	}
}
class Test
{
	public static void main(String args[])
	{
		new Hello(true);
	}
}

output 

 if pass arguments than it is execute
