throws

  • throws clause is exception handling mechanisum.
  • throws is overpass the compiler exception detecting.
  • it is handle Unchecked exception
  • throws pass is not professional way to handlign exception.

syntax 

returntype functionName(arguments,...) throws ExceptionClassName { //code }

example 

class Test
{
	public static void main(String args[]) throws Exception
	{
//		System.out.println(args[0]); // comipler unchecked exception

		char x = (char) System.in.read();

		System.out.println(x);
	}
}
