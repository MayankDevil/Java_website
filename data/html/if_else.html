if else statement

  • if else statement is controal structure statement that help to controal follow execute.
  • if expression return true so execute in if statmenet.
  • if condition is return true so execute first block esle execute second block.
  • ifelse statement is same like ternary operator.

syntax : if statement 

if (expression) { // true expression }

syntax : if else statement 

if (expression) { // true expression } else { // false expression }
