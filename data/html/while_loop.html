while loop

  • while loop is loop that first check condtion then execute.
  • it is hand that iteration than iteration depend upon user.
  • contion return ture then loop execute.

control flow

  1. check conditon.
  2. if condition return true execute inside block code.
  3. after end block go back to conditon check.
  4. loop iterative until to condition false.
  5. if condition is return false control execute out of while loop.

syntax 

while (condition) { //code }

example 

class Test
{
	public static void main(String args[])
	{

		boolean n = true;

		while (n)
		{
			System.out.println(n);
			n = false;
		}
		System.out.println(n);

	}
}
