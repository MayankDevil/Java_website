ternary operator

  • the opeator perform operation upon three operand that called ternary Operator.
  • if expression is return true so execute first expression other else.
  • it is very useful and short single if else code.

contorl flow

  1. the "condition" is an expression that evaluates to either ture or flase.
  2. if the condition is true, the value of "expressionIfTrue" is returned.
  3. if the condititon is false, the value of "expressionIfFalse" is returned.

syntax 

condition ? expressionIfTrue : expressionIfFalse ;

example 

class Test
{
	public static void main(String args[])
	{

		boolean n = (1 < 0)? true: false;

		System.out.println("the value is "+n);
	}
}

output 

the value is false
