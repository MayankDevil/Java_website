for loop

  • for loop is iterative statement.
  • in loop we are know how to stop execute condition initizalition.
  • for loop initiazlition condition itration set before.

control flow

  1. first check initialization.
  2. second check condition.
  3. execute code end to for loop closed bracket.
  4. return from closed bracket to condition.
  5. next goto iteration.
  6. execute code (3) repeat
  7. if condition return false loop control stop iteration.

syntax 

for (initialization; condition; iteration){ //code }

example 

class Test
{
	public static void main(String args[])
	{
		for (int i = 0; i < 5; i++)
		{
			System.out.println(i);
		}
	}
}
