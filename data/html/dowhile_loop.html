do while loop

  • dowhile loop is itertive loop but it run at once.
  • it check conditon after execute inner code.

control flow

  1. find do execute code.
  2. second find while check conditon.
  3. if condition true goto do against.
  4. if stop control and out of loop if condition is flase

syntax 

do{ //code }while(condition);

example 

class Test
{
	public static void main()
	{
		n = true;
		do
		{
			System.out.println(n);
			n = false;
		}
		while(n);
	}
}

output 

true
